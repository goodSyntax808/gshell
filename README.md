# GSHELL

gshell is a system shell written in rust. It is only designed to work with Unix.

## Features
 - Supports all operators (`;`, `&&`,  `&`, `||`, `|`, `>`, `<`, `>>`)
 - Executing programs
 - Single and double quote parsing
 - Command history (up/down arrow key)
 - Tab autocomplete
 - left/right arrow key
